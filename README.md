# Scan this Receipt
Python API to provide the receipt scanning functionality for Dun Buy More (https://github.com/ruiyigan/dunBuyMore)

# Tech
- PaddleOCR
- Flask
- Docker

# Run Locally using Docker
```
docker compose up --build
```
